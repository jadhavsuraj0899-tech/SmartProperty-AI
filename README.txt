🚀 Project Overview

SmartProperty AI is a Machine Learning-based desktop application that predicts real estate property prices based on user inputs such as carpet area, number of bedrooms, location, price per square foot, and amenities.

This project combines predictive modeling with an interactive GUI, making it easy for users to estimate property prices in real time.

🎯 Objective

To build an intelligent system that:

Analyzes property features
Predicts accurate real estate prices
Provides a user-friendly interface for non-technical users

✨ Features
📊 Multiple Linear Regression model for price prediction
🎚️ Slider-based input for carpet area (700–1500 sq ft)
🏠 Dropdown selection for bedrooms and location
📍 Location names mapped to encoded values
☑️ Checkbox selection for amenities
⚡ Real-time prediction with a single click
🖥️ Desktop GUI built using Tkinter
